- **Day 7:** Deploy to Vercel

## Figma ‚Üí HTML/CSS Translations

| Figma | HTML/CSS |
|-------|----------|
| Frame | `<div>` container |
| Auto-layout | CSS Flexbox/Grid |
| Fill/Stroke | `background-color` / `border` |
| Text properties | `font-size`, `color`, `line-height` |
| Spacing/padding | `padding`, `margin` |
| Corner radius | `border-radius` |
