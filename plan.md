## When You Get Stuck

### Week 2-3: "JavaScript feels too hard"
**This is NORMAL.** Most designers struggle here.
- Solution: Slow down, do more practice
- Use `console.log()` everywhere
- Repeat videos at 0.75x speed
- Take breaks - your brain is processing
- Remember: Senior devs Google syntax too

### Code Errors
1. Read error message carefully
2. Google: "[error message] react next.js"
3. Check Stack Overflow
4. Ask Claude Code (explain what's happening)
5. Check official documentation

### Feeling Overwhelmed
- Take a break (seriously)
- Build something fun, not from the plan
- Watch Fireship 10-min video (funny & informative)
- Extend timeline - strong foundation > rushing
- Remember your "why"
