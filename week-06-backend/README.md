# Week 6: Backend & Database

## Goal
Data persistence and user authentication. Build real applications with databases.

## What You'll Learn
- Database design (tables, relationships)
- CRUD operations (Create, Read, Update, Delete)
- User authentication and protected routes
- Environment variables and secrets

## Tools
- **Supabase** (Firebase alternative, SQL-based)
- **Prisma** (database ORM - easier than raw SQL)
- **NextAuth.js** (authentication)

## Projects
- **Day 1-3:** Notes app with database (create, edit, delete)
- **Day 4-7:** Bookmark manager with user accounts
  - Sign up / Login
  - Private bookmarks per user
  - Search and tags

## Video Resources
- Supabase crash course (6 hours)
- SQL basics (4 hours)
- Authentication flows (4 hours)
- Prisma ORM basics (4 hours)

## Key Concepts
- Databases and tables
- SQL queries (SELECT, INSERT, UPDATE, DELETE)
- User authentication flows
- Password hashing
- Session management
