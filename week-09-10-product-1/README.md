# Weeks 9-10: Product 1 - Design Tool for Your Domain

## Goal
Ship your first real product that solves a real problem.

## Choose ONE to Build

### Option A: Component Library Generator
- Upload Figma designs
- AI generates React components
- Download or copy code
- Tailwind CSS styling
- **Why:** Solves real problem for designers/devs

### Option B: Design System Analyzer
- Upload design file
- AI analyzes consistency
- Reports color/spacing/typography usage
- Suggests optimizations
- **Why:** Helps design teams maintain systems

### Option C: Rapid Wireframe Builder
- Drag-and-drop interface
- Pre-built component blocks
- Export to code or Figma
- AI layout suggestions
- **Why:** Speeds up design process

## Schedule
**Days 1-14:** Build, test with real users, iterate, launch

## Success Looks Like
- Working product
- 5-10 beta users tested it
- Deployed and live
- GitHub repo public
- Twitter post about launch
